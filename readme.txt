== Description ==

This is the 2StateReviews.com Wordpress Widget, which links the latest vote you cast to your blog. 2StateReviews is a review systems where things are either good or bad, no middle ground.


== Installation ==

Installation is straight forward.

1. Install in your plugins directory
2. Activate it in the plugins menu
3. Edit the 2StateReviews in Widget Menu to your 2StateReviews username and whatever you want the badge to be titled on your wordpress page.


== Frequently Asked Questions ==

= What is 2StateReviews =
